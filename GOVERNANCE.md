# Governance

SeaLegs is currently maintained as a small independent project.

## Maintainer

- Primary maintainer: DAWNCR0W

## Decision Making

The maintainer makes final decisions on:

- Product scope.
- Privacy and security boundaries.
- Release timing.
- Issue and pull request prioritization.
- Whether a contribution fits the maintenance budget.

Discussion is welcome, but maintainers may close issues or pull requests that
are out of scope, unsafe, too broad, or not maintainable.

## Project Values

- Privacy-first local processing.
- Clear user control.
- Minimal surprise around macOS permissions.
- Practical comfort improvements for 3D games.
- Maintainable native macOS code.

## Maintainer Availability

This project is maintained best-effort. Response times may vary.

## Becoming a Maintainer

Additional maintainers may be added later after sustained, high-quality
contributions and clear trust around privacy, security, and release discipline.
