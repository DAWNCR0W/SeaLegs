# Roadmap

This roadmap is directional, not a promise. SeaLegs will stay small and
privacy-first.

## Near Term

- Tune default profiles based on real game testing.
- Expand native-fullscreen and multi-display compatibility testing.
- Automate signed and notarized release packaging.
- Add profile import/export with validation and conflict handling.

## Mid Term

- Add more profile presets for common game categories.
- Improve session reports with clearer local recommendations.
- Add better fallback behavior for native fullscreen limitations.
- Add automated UI coverage for onboarding, Settings, and permission recovery.

## Later

- Explore optional import/export of user profiles.
- Explore a lightweight updater strategy.
- Explore advanced overlay themes while keeping the default quiet.
- Evaluate whether limited plugin-style profile packs make sense.

## Non-Goals

- Medical diagnosis or treatment.
- Cloud analysis of screenshots or video.
- Anti-cheat bypassing.
- Game memory patching.
- Gameplay automation.
- Support for Windows, Linux, or VR in this repository.
