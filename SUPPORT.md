# Support

SeaLegs is an independent open-source project. Support is best-effort.

## Before Asking for Help

Check:

- `README.md` for build, test, and local run commands.
- `docs/PRIVACY.md` for local data and permission behavior.
- `SECURITY.md` for private vulnerability reporting.
- Existing GitHub issues for known problems.

## Good Support Requests

Include:

- macOS version.
- Apple Silicon model.
- SeaLegs commit hash or release version.
- Whether the game is windowed, borderless, or native fullscreen.
- Whether Screen Recording permission is granted.
- Steps you tried.
- Expected result and actual result.

Do not attach private screenshots, raw diagnostic exports, or files containing
private paths unless a maintainer explicitly asks for a redacted version.

## Where to Ask

- Bugs: use the bug report issue template.
- Feature ideas: use the feature request template.
- Security or privacy vulnerabilities: follow `SECURITY.md`.
- General questions: use the question issue template.

## Not Supported

- Windows, Linux, VR, or non-macOS builds.
- Anti-cheat bypassing.
- Game memory patching.
- Medical diagnosis or treatment advice.
- Guaranteed compatibility with every native fullscreen game.
